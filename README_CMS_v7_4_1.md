# CMS v7.4.1 更新

- 作品資訊格式：中文名／英文名、年份／尺寸、類型／題材／材質／媒材；空值自動略過。
- 作品集與線上藝廊作品可點擊開啟防盜浮水印大卡片。
- 線上藝廊只顯示 `isGallery = TRUE`。
- 試算表換行會在頁面保留。
- 展覽經歷讀取「展覽經歷」分頁；歷史回顧讀取「歷史回顧」與「書籍」分頁。
- 39本戶口名簿等歷史資料改為站內卡片彈窗。
- 首頁輪播載入前不顯示虛構作品名稱。
- 品牌與 PWA 圖示改用 YING.ico（筆觀音）。
- Facebook 區使用官方嵌入。若「網站設定」可回傳 `facebookPostUrls`（最多三筆貼文網址），會形成三則橫向輪播；未提供時顯示粉專 Timeline。
