# CMS v7.2-D｜統一排序中心

## 本版完成

- 新增 `admin/sort.html` 統一排序中心。
- 四個展示區：首頁輪播、精選作品、線上藝廊、作品展示。
- 只顯示已在作品管理勾選的作品。
- 新勾選作品第一次進入排序中心時自動排在最後。
- 可直接拖曳，亦可使用上下按鈕。
- 儲存時自動重編 1、2、3……，不允許手填排序數字。
- 不同作者與作品庫共同排序。
- 新增試算表分頁 `網站展示配置`，由系統自動維護。

## 更新順序

1. 用新版 `admin/apps-script/Code.gs` 覆蓋 Apps Script。
2. 重新部署 Web App，版本選「新版本」。
3. 用本包全部內容覆蓋 GitHub 工作資料夾。
4. GitHub Desktop Commit / Push。
5. 開啟 `/admin/sort.html` 測試。

API 版本應顯示：`7.2-D-unified-sort`。
