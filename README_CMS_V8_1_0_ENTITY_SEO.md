# V8.1.0 Entity SEO（品牌權威）

一次完成：

1. 首頁品牌 SEO：Title、Description 以「謝秀英」為主體。
2. Person Schema：姓名、英文名、職業、照片、專長與官方網址。
3. sameAs：目前僅加入已確認的官方 Facebook，避免未驗證網址污染實體關聯。
4. Organization + WebSite Schema：加入品牌組織、Publisher、Author 與可實際運作的 SearchAction。

SearchAction 使用 `works.html?q=關鍵字`，作品集會依中文名、英文名、類型、題材、材質、媒材、年份與尺寸篩選。
