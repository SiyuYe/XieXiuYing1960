# 謝秀英書畫藝術館｜CMS v7.3-B 智慧媒材

## 本版完成

- 作品類型第一次設定時，可選擇是否套用預設材質與媒材。
- 只有作品類型真正變更時，才會再次詢問是否套用預設。
- 手動修改過的媒材不會被系統自行覆蓋。
- 媒材改為 Checkbox 多選，可同時選擇多項。
- 資料表、右側編輯面板、批次媒材修改皆支援多選。
- 新媒材字典：墨、礦岩、水干、膠、國畫顏料、水彩顏料。
- 陶藝改列為作品類型，不再列為媒材。
- 作品類型預設：
  - 寫意：宣紙＋墨、國畫顏料
  - 工筆：礬宣紙＋墨、國畫顏料、水干、膠
  - 膠彩：絹布＋礦岩、水干、膠
  - 書法：宣紙＋墨

## 更新步驟

### 1. Apps Script

用 `admin/apps-script/Code.gs` 完整覆蓋目前 Apps Script 的 `Code.gs`。

回到試算表重新整理後，執行：

`謝秀英藝術館後台 → ③-1 升級智慧媒材字典（不更動作品）`

這個動作只會：

- 重建媒材主資料字典
- 補齊作品類型的預設材質／媒材對照
- 補齊宣紙、礬宣紙、絹布等材質主資料

不會修改任何已存在作品的媒材、材質或其他作品資料。

接著重新部署 Web App：

`部署 → 管理部署作業 → 編輯 → 新版本 → 部署`

API 版本應顯示：`7.3-B-smart-media`。

### 2. GitHub

將本資料夾全部內容覆蓋到 GitHub 正式專案，Commit 建議：

`CMS v7.3-B smart media`

Push 後按 `Ctrl + F5`。

## 智慧保護規則

1. 類型未設定，第一次選類型：詢問是否套用預設。
2. 類型不變：永遠不會自動改媒材。
3. 媒材手動修改後：永久保留。
4. 類型真的變更：詢問是否套用新類型預設；按取消就保留目前媒材。
