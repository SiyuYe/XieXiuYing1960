# V8.0.8 作品放大卡片自適應修正

## 修改範圍
- 本月精選作品
- 線上藝廊
- 作品集

三處共用同一個 `openArtwork()` 放大卡片，因此同步套用相同顯示方式。

## 完成內容
- 移除放大卡片外層的上下、左右捲軸。
- 放大卡片自適應瀏覽器寬度與高度。
- 畫作依可用空間等比例縮小。
- 圖片維持 `width:auto` / `height:auto`，不超過圖片原始尺寸，不會為了填滿版面而強制放大。
- 桌機顯示圖片與資訊雙欄卡片。
- 手機改為圖片在上、資訊在下。
- 資訊過長時仍可捲動，但隱藏醜陋捲軸。
- 開啟作品時鎖定背景頁面，關閉時恢復。

- `/style.css`
- `/app.js`
- `/src/app.source.js`
- 根目錄前台 HTML（更新快取版本至 8080）
