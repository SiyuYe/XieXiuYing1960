# 謝秀英書畫藝術館 CMS V7.9.0 — STEP 8

本版本依施工順序完成「第 8 次：作品載入優化」。

## 本次完成

1. 作品集改為分批建立卡片。
   - 每批 16 件。
   - 首次只建立第一批。
   - 接近頁面底部時，支援 IntersectionObserver 的瀏覽器會自動加入下一批。
   - 同時保留「載入更多作品」按鈕，作為不支援 IntersectionObserver 的瀏覽器備援，也可供使用者手動操作。

2. 線上藝廊作品同樣採每批 16 件，避免一次建立全部作品卡片。

3. 作品分類切換時會：
   - 中止舊分類的觀察器。
   - 移除舊的載入控制。
   - 重新從第一批開始顯示新分類。

4. 首頁內容限制：
   - 精選作品只建立 8 件。
   - 首頁主輪播只使用 3 件作品。

5. 主輪播圖片載入：
   - 當前圖片使用 eager / high priority。
   - 只預載上一張與下一張，不預載全部作品。

6. 放大作品圖：
   - 2400 圖片只在使用者點擊作品、打開放大視窗後才設定 src。
   - 未點擊前不下載放大圖。

7. 圖片屬性：
   - 每個作品區的第一張使用 loading="eager"、fetchpriority="high"。
   - 其餘作品使用 loading="lazy"、fetchpriority="low"、decoding="async"。

8. 前端版本更新：
   - app.js?v=7908
   - style.css?v=7908
   - api/cms.js?v=7908（聯絡頁保留）
